HTML
HTML stands for Hyper Text Markup Language. It is widely used programming language in web developers. HTML has many tags which helps the developer to deploy their thought related to the web development. For the web developers, the designing part are included in CSS.
CSS
Without CSS the webpage seems to be like flower without smell. CSS is must for web developers to make good design for their webpage. CSS gives the nice touch to the webpage. It allows the developers to use different colors, text fonts, background images and so on which are almost impossible to do in HTML.
JavaScript
JavaScript is a scripting language which allows the user to create interaction on a webpage. It is usually for the form validation as it is aslo used to validate the form in this project. JavaScript is also used for various other reasons. And it is totally different from Java Programming language.
